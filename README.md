# Метеор Fit

Статическая версия сайта программы MeteorFit. Сделана на основе презентации и текущего сайта, без серверных ключей и приватных данных.

## Что внутри

- `index.html` — структура страницы.
- `styles.css` — внешний вид.
- `script.js` — мобильное меню и форма заявки через почтовое письмо.
- `assets/` — фотографии для сайта.

## Деплой

Сайт можно разместить на GitHub Pages, Vercel или Netlify. Для GitHub Pages достаточно загрузить эту папку в репозиторий и включить Pages из корня проекта.

Приватные SSH-ключи, токены и `.env`-файлы не должны попадать в репозиторий. Для этого в проект добавлен `.gitignore`.
